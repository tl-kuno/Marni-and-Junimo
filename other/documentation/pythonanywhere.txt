To update changes made to this code in the PythonAnywhere hosting
platform:

    First, make sure that requirements.txt is updated by running the 
    following in your terminal in this project directory:
        pip freeze requirements.txt

    Login to the site, you will be on the Dashboard
    Navigate to the Consoles Tab
    Open a bash console 
        cd Picnic-Quest
        git checkout [whatever branch you want to test funcitonality of]
        git pull
    Navigate to the Web Tab
    Click Green reload button
